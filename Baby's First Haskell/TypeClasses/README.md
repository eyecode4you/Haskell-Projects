## Common Type Classes
  + Enum
  + Eq - Equality
    + == and /=
  + IO - IO actions
  + Integral - For integers and similar
  + Num - Numeric types
  + Ord - types that have ordering
    + e.g. compare, <, <=, max, min, etc.
  + Read - types that can be read from strings
  + Show - for types which we can generate a string representation
  + Functor
  + Monad
  + Monoid
